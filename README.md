# Application Bancaire avec Protocole SET

## Description
Implémentation complète du protocole SET (Secure Electronic Transaction) pour une application web bancaire sécurisée.

## Architecture
- **Frontend**: HTML/CSS/JavaScript avec Bootstrap
- **Backend**: Flask (Python)
- **Cryptographie**: RSA 2048, signatures digitales, certificats X.509
- **Interface**: 3 rôles (Client, Banque, Autorité de Certification)
